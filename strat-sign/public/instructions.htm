Instructions for the Signal Sharing Game

Please, read them carefully!


This game is played by one human player and two automated player.

In each round, you and the automated players have to decide as a group whether an urn is blue or red. Both colors are equally likely.

In the beginning, each player receives between 0 and 3 private signals. This means, that no other player has the same signal. There is a % chance that a signal contains a color and with a chance of % it is empty and not shown.
If a signal shows a color, there is a % chance that the color shown is the same as that of the urn and a % chance that the color is different.
In the first part of the game you have to decide whether you want to share any single signal with the automated players or hide it from them.
The automated players always share all of their signals.

Next, you see all the signals each player deciced to share and you have to make a voting decision that indicates your belief of the true color of the urn.
The automated players receive all signals that were shared too. Then they make their voting decion in the following way:
1. They count the number of shared signals that show 'red' and multiply it by .
2. Then they count the number of shared signals that show 'blue' and multiply if by .
3. Finally, the automated players compare the numbers they received and vote for the color that has the larger number. If both number are the same, they decide for 'red'.

In the end all the votes for 'blue' and 'red' are counted and the color with more votes is the group decision. If the decision has the same color as the true color of the urn, you earn 2 USD.

The game is repeated a certain number of rounds, and each round the color of the urn might change.

If you understood the instructions correctly press the "DONE" button to proceed to the game.
